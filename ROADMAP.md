# YouTube AI DJ Roadmap

## Phase 1

- Keep the current AI DJ visual shell
- Search YouTube videos safely
- Play embeddable videos in the main media slot
- Save queue, history, and vibe notes locally

## Phase 2

- Add backend-assisted search ranking
- Add account-backed sync for on-the-go use
- Add mobile layout tuning for iPhone
- Add deploy target instead of local-only hosting

## Phase 3

- Generate fallback visuals when no official music video exists
- Add job queue, render history, and confidence badges
- Separate official video, fan upload, and AI-generated video states

## Product boundary

- Spotify edition: Spotify playback and personal listening
- YouTube edition: public/shareable video-first AI DJ
